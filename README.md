## ✨ 主な機能

### 👥 複数人対応
- 1～10人まで同時にスコア管理可能
- カードゲーム、ボードゲーム、eスポーツなど様々なゲームに対応

### ⚡ リアルタイム表示
- スコアを入力するたびに合計が自動更新
- ゲーム途中でも現在のスコアが一目でわかる

### 📊 スコア倍率設定
- 最終スコアに倍率を適用可能
- 複数ゲーム実施時に最終スコアを自動計算
- 例：1勝 = 100点 × 2倍 = 200点

### 🏆 自動ランキング
- ゲーム終了時に自動的に順位を表示
- 誰が勝ったかひと目瞭然

### 📱 モバイル対応
- スマートフォン・タブレット完全対応
- レスポンシブデザイン対応

### 🔐 プライベート
- すべてのデータはブラウザのローカルストレージに保存
- サーバーへの送信なし
- プライバシー完全保護

### 🔌 オフライン対応
- インターネット接続なしで使用可能
- ブラウザだけで動作

## 🎯 使用例

### 🃏 カードゲーム「大富豪」
```
プレイヤー: 4人
スコア: 1位=100点、2位=50点、3位=20点、4位=-100点
倍率: 3倍（3ゲーム実施予定）

各ゲームのスコアを入力 → 合計が自動計算 × 3倍で表示
```

### 🎲 ボードゲーム
```
プレイヤー: 3人
複雑なスコア計算も自動で処理
```

### 🎮 eスポーツ大会
```
プレイヤー: 複数チーム
大会用スコア管理に最適
```

### 🏅 スポーツリーグ
```
ダーツ、ビリヤード、ボウリングなどのスコア管理
リーグ戦の管理に対応
```

## 🚀 使い方（3ステップ）

### 1️⃣ プレイヤー設定
```
1. プレイヤー数を入力（1～10人）
2. 各プレイヤーの名前を入力
3. 「ゲーム開始」をクリック
```

### 2️⃣ スコア記入
```
1. スコア倍率を設定（デフォルト：1倍）
2. 各プレイヤーのスコアを入力
3. 「スコア記録」をクリック
4. リアルタイムで合計が更新される
```

### 3️⃣ 結果確認
```
1. 「ゲーム終了」をクリック
2. ランキングが自動表示
3. 勝者が一目瞭然！
```

## 💻 開発セットアップ

### 前提条件
- Node.js 16以上
- npm 7以上

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/yakinorinori/Game-score.git
cd Game-score

# 依存関係をインストール
npm install
```

### 開発コマンド

```bash
# TypeScriptをコンパイル
npm run build

# ウォッチモード（自動コンパイル）
npm run watch

# ローカルサーバーで実行
npm start
```

### ブラウザでアクセス
```
http://localhost:8000
```

## ❓ よくある質問（FAQ）

### Q1: 登録は必要ですか？
**A:** いいえ。登録不要で、すぐに使い始めることができます。

### Q2: オフラインで使用できますか？
**A:** はい。ブラウザさえあればオフラインでも利用可能です。

### Q3: データは保存されますか？
**A:** はい。ブラウザのストレージに自動保存されます。次回アクセス時に履歴を確認できます。

### Q4: 最大何人まで対応しますか？
**A:** 1～10人までに対応しています。さらに多い人数の場合はご相談ください。

### Q5: どのブラウザで動作しますか？
**A:** Chrome、Safari、Firefox、Edge など、最新ブラウザであればすべて対応しています。

### Q6: データは外部に送信されますか？
**A:** いいえ。すべてのデータはブラウザ内に保存され、外部には一切送信されません。

### Q7: スマートフォンで使用できますか？
**A:** はい。スマートフォン・タブレット完全対応です。タッチ操作で快適に利用できます。

### Q8: CSV出力できますか？
**A:** 現在は未対応ですが、今後の更新で対応予定です。

## 🐛 トラブルシューティング

### 問題：スコアが表示されない

**解決方法:**
- ブラウザをリロード（F5キー）
- キャッシュをクリア
- デベロッパーコンソールでエラーを確認

### 問題：データが保存されていない

**解決方法:**
- ブラウザのストレージが有効か確認
- シークレットウィンドウではなく通常ウィンドウで開く
- ブラウザのプライベートブラウジングモードをオフに

### 問題：画面が表示されない

**解決方法:**
- ブラウザの互換性を確認（最新版を使用）
- キャッシュをクリアして再読み込み
- 別のブラウザで試す

### 問題：レイアウトがおかしい

**解決方法:**
- 画面の向きを変更（縦向き ↔ 横向き）
- ズームレベルをリセット（Ctrl+0）
- ブラウザの拡張機能を一時的に無効化
